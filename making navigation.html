Making bottons with Fireworks

This is the navigation bar made up of bottons created with Fireworks

Steps to make bottons

  1. set up your canvas size according to the size of your navigation bar
  2. draw your botton (usually rectangle)
  3. with the filter option, add an "Inner Bevel" filter to create a look of an botton
  4. save it as an symbol
  5. drag the number of botton needed for the navigation bar
  6. edit the botton by double clicking it
  7. change how it looks when it is over and under (copy up image than change filter option)
  8. align the bottons evenly
  9. trim your canvas to make it a perfect fit to crop unnecessary white spaces around the botton

Using the bottons in Dreamweaver

  1. Save the image as a html in dreamweaver with the "export" option
  2. In a tempelete document, create an div at the place you want your bottons to show on your page.
  3. Click on insert on the top navigation bar and choose "Fireworks html".
  4. Select your botton file and it should show up in the div you created
  5. Then link your bottons to pages on your file
  6. To recreate the hover effect, first copy the <script> code shown in the browser preview into the head of the tempelete.
  7. Then change the link to the images by removing the ../ in the "a herf=" link (use the find and replace option).
